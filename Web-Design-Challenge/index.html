Summary: Latitude vs. X


This application contains analysis, represented visually, of weather behavior in contrast to distance from the equator. The data used in this analysis was sourced from "OpenWeatherMap API", which includes over 500 cities. The features of weather used were wind, humidity, temperature, and cloudiness. In addition to the visualizations, this application includes a data page that holds all of the source data used for the analysis.

This analysis yielded a couple of note worthy details, one being the lack of linear correlation between latitude and cloudiness, the other being the relationship between the decrease in latitude and incease in temperature.
